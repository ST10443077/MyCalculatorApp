# MyCalculatorApp

A simple Android calculator application built with **Android Studio** and **Kotlin**.  
This project demonstrates basic arithmetic operations, clean UI design, and GitHub integration for version control.

---

## 📱 Features
- Perform basic arithmetic: addition, subtraction, multiplication, division
- Clear and reset functionality
- Responsive layout for different screen sizes
- Built using modern Android development practices

---

## 🛠️ Tech Stack
- **Language:** Kotlin
- **IDE:** Android Studio
- **Build System:** Gradle
- **UI:** XML layouts with Material Design components

---
